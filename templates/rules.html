Software Contest Rules & Regulations

Event Details

  • Individual registration only - Team registrations are not allowed
  • The event consists of two rounds: Preliminary Round and Final Round

Preliminary Round

  • Format: Multiple Choice Questions (MCQs) based on Java
  • Duration: 1 hour
  • Top 20 participants will qualify for the Final Round

Final Round (Coding Round)

  • Participants will be provided with code containing minor errors
  • The task is to debug the code to pass the given test cases
